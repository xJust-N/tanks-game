# Семестровый проект №2 студента 2 курса Котова Никиты из 11-401

Проект представляет собой игру танчики, можно играть вдвоем локально, или с любым количеством игроков по сети на сервере

## Краткое описание
 - Возможность игры локально или по сети
 - Управление танком: езда, повороты, стрельба
 - 2 вида орудий: обычное и ракетное
 - 3 вида усилений: скорости, здоровья и пушки

## Использованные технологии
 - **Java** - язык программирования
 - **Selector** и **NIO** на сервере для обработки всех клиентов в одном потоке
 - **Swing** для ui на клиенте, так же был использован **Graphics2d** для отрисовки компонентов карты
 - Логирование **SLF4J + Logback** на клиенте и на сервере, в классах для чтения и записи в каналы

## Описание протокола
 1) Клиент отправляет приветственное сообщение с именем
 2) Сервер регистрирует и отправляет карту со всеми объектами
 3) Далее клиент будет отправлять команду 1 байтом, сервер будет ее выполнять
 4) Сервер отправляет всем клиентам обновления происходящие в мире:
    * перемещение объекта(тип сообщения + id объекта + направление + координаты)
    * создание нового объекта или обновление текущего(тип сообщения + весь объект)
    * удаление объекта(тип сообщения + id)
    * конец игры в случае когда на карте остался только 1 игрок(только тип)
      (все типы сообщений приведены в enum ChannelMessageType)
